# Contributing to PixelMap

Thank you for your interest in contributing to PixelMap!

## Ways to Contribute

### Bug Reports

Found a bug? Please [open an issue](https://github.com/m-beau/channelmap_generator/issues) with:

- A clear, descriptive title
- Steps to reproduce the issue
- Expected vs. actual behavior
- Your environment (OS, Python version, browser if using GUI)

### Feature Requests

Have an idea? [Open an issue](https://github.com/m-beau/channelmap_generator/issues) describing the problem and your proposed solution.

### Pull Requests

For significant changes, please open an issue first to discuss your approach.

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature-name`
3. Make your changes and run tests: `pytest`
4. Push and open a pull request

## Code Style

We use [Ruff](https://github.com/astral-sh/ruff) for linting and formatting. Run `ruff check .` and `ruff format .` before submitting.
